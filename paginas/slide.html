Carrossel de Elementos

Construindo um carrossel de elementos com a biblioteca wd

Apresentação

O objetivo deste tutorial é descrever a construção de um carrossel de elementos cujo propósito é exibir um elemento filho de cada vez na tela, seja de forma manual ou automática.

O manual da biblioteca wd pode ser encontrado no link .

Elementos Básicos

Para demonstrar a construção do carrossel, será estabelecido um container pai contendo quatro elementos filhos, conforme abaixo apresentado:

Exibição Manual

Nesta fase serão contruídos os elementos de interação para exibir cada slide de forma individual.

Para isso, cada elemento filho será contemplado com um identificador (id) e será necessário contruir um navegador para manipular o slide a ser exibido. Esse navegador possibilitará escolher o elemento específico e também o anterior ou posterior:

Slide 1
Slide 2
Slide 3
Slide 4
1 2 3 4 
div class=wd-b wd-size-xlarge wd-align-center
	div style=height: 150px; id=slide1 class=wd-bg-blue  Slide 1/div
	div style=height: 150px; id=slide2 class=wd-bg-red   Slide 2/div
	div style=height: 150px; id=slide3 class=wd-bg-green Slide 3/div
	div style=height: 150px; id=slide4 class=wd-bg-black Slide 4/div
/div

nav
	span class=wd-icon-w /span
	span1/span
	span2/span
	span3/span
	span4/span
	span class=wd-icon-e /span
/nav
